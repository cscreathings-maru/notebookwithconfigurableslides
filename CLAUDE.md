# NotebookLLM-custom — Spec-Driven Development project

This project uses GitHub Spec Kit. The spec-driven workflow lives in `.specify/`:
- `.specify/templates/` — spec, plan, tasks, checklist, constitution templates
- `.specify/scripts/bash/` — helper scripts used by the workflow commands
- `.specify/memory/constitution.md` — project principles (created by the constitution step)
- `specs/<NNN-feature>/` — generated spec.md, plan.md, tasks.md per feature

Workflow command instructions are in `.claude/skills/speckit-*/SKILL.md`.
